Hello, World!

Welcome to the Open Source Club at UT Dallas!! Our goal is to build a strong open source community on campus!

About

Open Source Club? Doesn't UTD already have a linux club??

Yes, UTD does have a linux users group, and they're amazing!
But we're about Open Source Development! While Open Source is often associated with Linux, it's wayyy bigger than just an Operating System.
We want to introduce students to real-world software development using industry relevant tools!
For Fall 2019 we're focusing on web development using open source technologies.

Oh okay, so you're doing web design?! Like a UI/UX club?

Ummm...Not Exactly. Although web design is an integral part of web development, we're going to cover all aspects of web development , which includes front-end and back-end web development.
The skills we're going to teach will help students with hackathons, personal projects, resume etc.

Wait, how're you guys going to teach real-world software development? Don't the CS courses at UTD cover that?

College level CS classes focus on logic and problem-solving, in order to build a strong foundation for long-term success.
But the tech industry moves very rapidly in terms of tools being used and technologies being adopted, just having a good GPA isn't enough!
Our philosophy is that people learn best by doing. So we'll have plenty of projects for our members to work on, and If you have an idea that you want to implement, you can form a team and start coding right away! We support and encourage collaboration between students on projects.

What kind of projects will I be able to work on? I don't wanna implement another inventory system in C++/Java

Web page redesign for UTD, making responsive web pages, making a fully functional blog application, playing with awesome API's, and much more. There's no limit to what you can create given the right tools!

This is all so intimidating... I'm not sure if I'm ready for something like this

How will you know if you don't try?
Our goal is to build a strong open source community on campus along with a culture of collaboration!
Everyone, {student, faculty, staff}, is welcome at our events!
We especially encourage participation from freshmen and sophomores students! Come over to get a taste of what you'll be doing in a software internship/job!
